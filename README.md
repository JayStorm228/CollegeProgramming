# Основы Алгоритмизации и Программирования

Этот репозиторий содержит работы студентов по предмету **Основы Алгоритмизации и Программирования**. Здесь хранятся выполненные задания (включая мои собственные как примеры), блок-схемы и учебный материал.

**Важно для студентов:** Мои работы — это ориентир и примеры для понимания. **Не используйте их напрямую** — преподаватель легко заметит плагиат! Репозиторий публичный для удобства хранения и обмена, но копируйте идеи, а не код целиком.

## 📁 Структура репозитория

Основы Алгоритмизации и Программирования/  
├── Code/  
│ └── <Фамилия_студента>/  
│ └── <Номер_темы>/  
│   ├── Ex_1.py  
│   └── Ex_2.py  
├── Schematics/  
│ └── <Фамилия_студента>/  
│ └── <Номер_темы>/  
│ ├── <Номер_темы>.drawio  
└── Textbook.pdf

- **Code/**: Все решения на **Python**. Задачи — упражнения из учебника (сортировки, циклы, рекурсия и т.д.).
- **Schematics/**: Блок-схемы к коду (в формате .drawio или PNG).
- Папки по фамилиям студентов (латинницей) для удобства. Темы нумеруются по учебному плану.

**Пример пути к файлу:** `Code/Ivanov/2.1.1/Ex_1.py`
